# Update Database

## Update Database from Dev Environment

- change directory to your senar folder
- run script .\db.ps1

```bash
.\db.ps1
```

- type 1 from menu and press enter

```bash
=========== Senar Database ===========
1: Get senar.sql from latest db dev
2: Get senar.sql from latest db prod
3: Load senar.sql to local database
4: Remove senar.sql from current directory
q: Press 'q' to quit.
======================================
Please select: 1
```

- wait until senar sql downloaded

```bash
Getting senar.sql from latest db dev
senar.sql    100%  201MB   3.9MB/s   00:51
Download completed as senar.sql
```

- type y at prompt and press enter

```bash
Also load senar.sql dev to local database? [y/n]: y
```

- wait until database is loaded to local database

```bash
Loading senar.sql to local database
Loading Completed
```

- type y and enter to remove unnecessary senar.sql file

```bash
Also remove senar.sql from current direcorty? [y/n]: y
```

- press enter to back to main menu

```bash
Remove senar.sql from local directory
Remove Completed
Press Enter to continue...:
```
